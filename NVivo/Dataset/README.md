# README

## Title
Sample_Dataset

## Description

This dataset is adapted from the U.S. Environmental Protection Agency’s _Data Center Energy Efficiency Focus Groups and Interviews_ dataset. A subset of files was selected for the NVivo tutorial and renamed for clarity and consistency, while file contents remain unchanged.

---

## Data Source
Original dataset: U.S. Environmental Protection Agency. _Data Center Energy Efficiency Focus Groups and Interviews_. Available via Data.gov: https://catalog.data.gov/dataset/data-center-energy-efficiency-focus-groups-and-interviews
